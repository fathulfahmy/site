The Barter App was created to facilitate safe and efficient service exchanges between users. The project included service search and request functionality, in-app messaging, payment processing, statistical tracking, fraud reporting, account suspension, and an admin panel, providing a reliable, intuitive platform for both users and administrators. Before this project, peer-to-peer service exchanges lacked structure, security, and administrative oversight, creating inefficiencies and trust issues. My goal was to develop a unified, mobile-first platform with comprehensive backend support, ensuring usability, security, and operational clarity were central. The final result is a fully integrated mobile and admin ecosystem, combining React Native, Laravel, and detailed API documentation to deliver a scalable, secure, and user-friendly service exchange solution.
